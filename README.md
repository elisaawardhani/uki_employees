
# UKI Employees

An Application for managing UKI employees data.




## Installation



How to Run:

1. Clone the project


```bash
  git clone https://github.com/elisaawardhani/uki_employees
```

2. Run Backend:

```bash
  cd backend
  npm install
  add .env file
  npm run start
```

3. Run Frontend:
    
    Make sure your computer already has Angular 6 installed.

```bash
  cd frontend
  npm install
  ng serve
```
